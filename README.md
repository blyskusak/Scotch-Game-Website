# Scotch Game Website

This repository contains a simple, static website dedicated to the Scotch Game chess opening. It was created as a university project for the **AWWW** course.

## 📖 About the Project
The main goal of this project is to demonstrate basic web development skills using core front-end technologies. The website serves as a straightforward informational page, presenting various lines and gambits within the Scotch Game using text, static images, and `.gif` animations for chess move visualization.

## ♟️ Website Structure
The project consists of a main page and several subpages covering specific variations:
* **Classical Variation**
* **Schmidt Variation**
* **Malaniuk Variation**
* **Scotch Gambit**
* **Göring Gambit**

## 🛠️ Technologies & Features Used
* **HTML5** – Used for semantic page structure and multi-page navigation.
* **CSS3** – Applied for visual styling, layout formatting, and responsive design elements.
* **Static Assets** – Integration of `.png` images and `.gif` animations to effectively illustrate chessboard positions and move sequences.
* **GitHub Pages** – Used to deploy and host the live version of the project.
